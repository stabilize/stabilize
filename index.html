Technology

Network Stabilization

  • connect Stabilized servers and desktops in secure, isolated networks
  • replicate or import some or all machines and services in each network (DNS, DHCP, IDP provider)
  • take advantage of the Stabilize preconfigured emulated network services for DNS, Active Directory, DHCP, etc.
  • optionally provide secure live-internet access to Stabilized networks or access to archived websites within a Stabilized network at the original URLs

Data Transfer

  • securely transfer data in and out of Stabilized networks
  • import machine disk images for conversion to Stabilized hardware
  • use Stabilize's secure temporary storage area to upload data for access in Stabilized machines
  • export content from a Stabilized machine to an attached drive, browse the contents in the Stabilize UI and download files as needed

Stabilize Secure Connector

  • the Stabilize Secure Connector (SCC) opens a secure connection between your local machine (or your corporate network) and a Stabilized machine
  • use local applications via the SSC to access Stabilized machines' APIs and interfaces (e.g. RDP, FTP, SQL, SSH) to send and receive data in bulk
  • the SSC can be used in your corporate network to enable the use of legacy URLs and IP addresses for Stabilized machines for all in-network users

Global PDF Printer

  • print from any stabilized application
  • enable print to PDF as an option for specific networks, machines, or end users
